Min Max element in array


Problem statement -

Write a program in any language to find the maximum and minimum element in an array.

Code

#include <iostream> int main() { int arr[5]; for (int i = 0; i < 5; i++) { cin >> cin[i]; } int min = max = arr[0]; for (int i = 1; i < 5; i++)) { if (arr[i] < min) min = arr[i]; else if (arr[i] > max) max = arr[i]; } cout << min << " " << max; }
Sample input

21 6 19 1 -5

Sample output
-5 21
Variables used

int array, int iterator i, min, max
